# iperf

iperf 是一个网络测速工具。

## CPU 占用对比

```bash
# 开启硬加速（设置 500M 带宽）
iperf -c 50.50.111.6 -u -i 1 -b 500m -l 1024 -t 999999 -p 2023

# 开启软加速（设置 500M 带宽）
iperf -c 50.50.111.6 -u -i 1 -b 500m -l 1024 -t 999999 -p 2023

# 关闭加速（设置 500M 带宽）
iperf -c 50.50.111.6 -u -i 1 -b 500m -l 1024 -t 999999 -p 2023
```
