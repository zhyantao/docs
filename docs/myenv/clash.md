# Clash

> ⚠️ **重要提示**：Clash 核心项目已于 2023 年底停止维护。以下客户端均为基于 Clash 内核的第三方图形界面工具，部分已迁移至新内核（如 Mihomo）。建议用户关注项目维护状态，谨慎选择。

Clash 是一个开源的跨平台网络代理工具，支持多种代理协议（Shadowsocks、VMess、Trojan 等），通过规则分流技术实现智能流量管理。以下是目前主流平台可用的客户端推荐。

---

## Windows 平台

### 1. Clash Verge Rev（推荐）

**特点**：基于 [Mihomo](https://github.com/MetaCubeX/mihomo) 内核的现代化客户端，继承 Clash 功能并持续更新，界面简洁，支持订阅管理、规则分流。

**下载**：[前往下载](https://dl.haojichang.com/apps/clash-verge/)

### 2. Clash for Windows（已停更）

**特点**：功能全面的经典图形客户端，支持订阅管理、规则分流、流量统计。**注意：该项目已停止维护，最后版本为 0.20.39**。

**下载**：[前往下载](https://dl.haojichang.com/apps/clash_for_windows/)

## iOS 平台

Clash 官方未推出 iOS 版本，但以下第三方客户端支持 Clash 规则格式：

### 1. Shadowrocket

**特点**：iOS 平台最流行的代理工具，支持 Clash 规则订阅，社区教程丰富，适合新手。**付费应用**（约 $2.99）。

**下载**：[App Store](https://apps.apple.com/us/app/shadowrocket/id932747118)

### 2. Stash

**特点**：专为 iOS 设计，支持 Clash 规则与订阅，界面现代化，支持高级分流规则。**付费应用**。

**下载**：[App Store](https://apps.apple.com/us/app/stash-rule-based-proxy/id1596063349)
